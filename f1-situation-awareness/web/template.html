Pagina Principale (Home)

Benvenuto nella dashboard di F1 Situation Awareness. Utilizza il menù in alto per navigare tra le varie sezioni di analisi dei dati telemetrici e di gara.

Come creare le altre pagine del progetto:

  1. Salva questo file come index.html.
  2. Fai 5 copie di questo file nella stessa cartella.
  3. Rinomina i nuovi file esattamente come indicato nel menù: risultati_finali.html, passo-gara.html, stint-gomme.html, telemetria-singola.html e confronto-telemetria.html.
  4. Trucco per il menù: Apri ogni file e sposta la classe w3-teal sul link corrispondente (ad esempio, in passo-gara.html, togli w3-teal da Home e mettilo sul link di Passo Gara) per evidenziare in quale pagina si trova l'utente!
